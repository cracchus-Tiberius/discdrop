# 3. BRAND RELATIONSHIPS & MANUFACTURING

## 3.1 Ownership Groups

### Trilogy (Latitude 64 / Dynamic Discs / Westside Discs)
- All manufactured in **Latitude 64's factory in Skellefteå, Sweden**
- Dynamic Discs (Kansas, USA) is the distribution/marketing arm
- Westside Discs (Finland) provides designs
- Latitude 64 handles all production
- **Plastics are identical** across brands, just named differently
- As of 2025–2026, Westside Discs products are transitioning away from the Latitude 64 store

### MVP Family (MVP / Axiom / Streamline)
- All owned by **MVP Disc Sports (Marlette, Michigan)**
- MVP = flagship, black overmold rim
- Axiom = colored overmold rim, related molds to MVP
- Streamline = single-mold (no overmold), lower price point
- **All share identical plastics**

### Innova Family
- Innova is the original major disc golf manufacturer (founded 1983)
- **Discmania's "Originals" line** was historically manufactured by Innova (molds like PD, FD, DD, TD, MD, P2)
- Discmania's "Evolution" line is manufactured by Latitude 64
- Discmania has been transitioning to self-manufacturing in recent years
- **Millennium** uses Innova-manufactured molds

### Prodigy
- Independent manufacturer (Carrollton, Georgia)
- European distribution formerly by Paju family's Prodigy Europe (since ~2014)
- As of 2026, **Powergrip** (Finland) acquired the European distributor role

- **Max weight** = heaviest available for that mold
- **Lightweight / Light** = typically under 165g

# 8. SCRAPER MATCHING TIPS

## 8.1 Common Challenges

1. **Plastic name variations**: "Champ" vs "Champion", "ESP" vs "E.S.P.", "K1 glow" vs "K1Glow"
2. **Player names in product titles**: "Sexton Firebird" = Innova Halo Star Firebird (Nate Sexton Tour Series)
3. **Old vs new names**: "K2" = "K1 Soft" (Kastaplast), "350G" = "300 Firm" (Prodigy)
4. **Abbreviated disc names**: "TB" = Thunderbird, "FB" = Firebird, "FD" = Fairway Driver (Discmania)
5. **Norwegian language**: Product descriptions may be in Norwegian
6. **Weight in product name**: "Destroyer 175g" — extract weight separately
7. **Color in product name**: "Blue Star Destroyer" — extract color separately
8. **Multiple brands with same model name**: Unlikely but check brand first

## 8.2 Regex Patterns for Common Formats

```
# Innova pattern
/^(Innova\s+)?(Halo\s+Star|Star|Champion|GStar|DX|Pro|KC\s*Pro|Blizzard|Nexus|XT|R-Pro|Color\s+Glow|Metal\s+Flake|Echo\s+Star|Proto\s+Glow|Halo\s+Champion|Halo\s+Nexus)\s+(.+?)(\s+\d{2,3}g?)?$/i

# Discraft pattern
/^(Discraft\s+)?(Z\s+Swirl|Z\s+FLX|Z\s+Glo|Big\s+Z|CryZtal\s+FLX|CryZtal\s+Glo|CryZtal|ESP\s+FLX|ESP\s+Glo|ESP|Titanium|Ti|Z|Jawbreaker|Putter\s+Line|Pro-D|Rubber\s+Blend)\s+(.+?)(\s+\d{2,3}g?)?$/i

# Kastaplast pattern
/^(Kastaplast\s+)?(K1\s+Glow|K1\s+Soft|K1\s+Hard|K1\s+Grind|K1|K3\s+Hard|K3\s+Glow|K3|K4\s+Glow)\s+(.+?)(\s+\d{2,3}g?)?$/i

# Prodigy pattern
/^(Prodigy\s+)?(200|300\s+Soft|300\s+Firm|300|400G|400|500|750G|750|AIR|Special\s+Blend)\s+([A-Z]{1,2}\d)(.+)?$/i
```

## 8.3 Flight Number Standard

All brands use the 4-number system: **Speed / Glide / Turn / Fade**
- **Speed**: 1–14+ (how fast the disc needs to be thrown)
- **Glide**: 1–7 (how well the disc stays in the air)
- **Turn**: +1 to -5 (high-speed stability; negative = turns right for RHBH)
- **Fade**: 0–5 (low-speed stability; how much it hooks left at end for RHBH)
